Brief Explanation of Floats

A floated element will be taken out of normal flow and will float to the left or the right, resting against one of three things:

  1. The edge of the viewport
  2. The edge of the parent containter
  3. The edge of another floated item

A float element has 4 possible values

  1. img { float: left; }
  2. img { float: right; }
  3. img { float: none; }
  4. img { float: inherit; }

How do floats interact with other elements?
